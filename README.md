# 标尺（Ruler.js） {#ruler}

## 简述 {#what-ruler}

使用原生canvas绘制标尺，可以进行平移与缩放，适用于低代码平台作为背景画布，也可以以此基础制作2d图形编辑器，或与`Three.js` 结合制作场景编辑器

由于是js，所以不受前端框架限制，适用于各个项目中
